[INGLISH/INGLES]:
The missions are established in each good song, these missions give you points
that you can use in future things. These points are not reset when installing another version

To add a mission you must enter the song and choose which mission you want to enable

Currently there are only 5 selectable missions:

"mision1": This Mission is for you to complete the song
"mision2": This Mission is for you to press the special arrows (if you fail 1 the mission automatically fails)
"mision3": This Mission is that you have to avoid missing more than 5 arrows
"mision4": This mission is to have a ranking above 50%
"mision5": This mission is that you have to keep your life below 25% at the end of the song

These are all the missions available in version 2.0. more will be added soon

The score added to each mission is random but difficult missions have more points.

Modifying it so that it only executes some missions will only give you points from that mission

but Mission 1, if you have already completed the song 1 time, will no longer give you more points.

This system is in Beta and points don't do much yet but several functions will be added in the future

Thanks for reading and Enjoy V2.0 of Ending Corruption
